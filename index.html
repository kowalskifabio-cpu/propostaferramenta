PROPOSTA COMERCIAL

Aplicativo para Conversão de Arquivos de Plano de Saúde

Leitura, tratamento e geração de arquivo compatível com importação no sistema Questor.

Projeto desenvolvido para Escrita Contabilidade com foco em automação operacional, padronização e redução de retrabalho.

Ver proposta
01

Contexto do Projeto

A Escrita possui demanda operacional na leitura de arquivos de retorno de plano de saúde exigindo tratamento manual para integração ao Questor.

02

Solução Proposta

Desenvolvimento de aplicação para leitura automática, tratamento e geração de arquivo compatível.

03

Escopo Incluso

  • Upload arquivos
  • Conversão automática
  • Exportação Questor
  • Testes
  • Treinamento
04

Cronograma

Semanas 1–2 → Levantamento

Semanas 3–5 → Desenvolvimento

Semanas 6–7 → Testes

Semana 8 → Entrega

05

Investimento

2 parcelas de R$ 3.000

06

Garantia
